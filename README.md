CAD & Automation

This repository focuses on CAD-related automation work for the Garment Industry.

Scope of work:
- Excel-based automation for fabric consumption
- Marker efficiency calculation
- Size-wise and style-wise production summaries
- Daily and monthly auto reports
- CAD support data for Gemini CAD & Optitex users

Purpose:
To reduce manual work, save time, and improve accuracy in CAD and production planning.

Note:
This repository contains sample files, formats, and logic references.
Actual production data and buyer-specific information are not shared.
